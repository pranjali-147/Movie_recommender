# üé¨ Bollywood & Hollywood Movie Recommendation System

A full-stack movie recommendation web application built using **Flask**, **Python**, and **Machine Learning**.  
The system provides intelligent movie recommendations using **content-based filtering** and **cosine similarity**, with separate modules for **Bollywood** and **Hollywood** movies.

The application displays the **Top 30 most similar movies**, sorted in **descending order of release year**, and dynamically fetches movie posters using the **OMDb API**.

---

## üöÄ Features

### üé• Bollywood Movie Recommender
- Recommend movies similar to a given movie
- Search movies by:
  - Actor
  - Director
  - Genre
  - Release Year
- Displays **Top 30 movies** in **descending order of year**
- Dynamic posters & metadata via OMDb API

### üé¨ Hollywood Movie Recommender
- Movie recommendation based on content similarity
- Displays **Top 30 similar movies**
- Sorted in **descending order of release year**
- Simple and fast recommendation flow

---

## üß† Recommendation Logic

- **Content-Based Filtering**
- **Cosine Similarity Matrix**
- Pre-trained similarity models stored using **Pickle**
- Instant recommendations without retraining

---

- **Backend:** Flask, Python
